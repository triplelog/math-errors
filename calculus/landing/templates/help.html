How to use the calculator

  1. Enter the function you want to take the derivative of in the box to the left and click See Solution.
  2. Check that the function that appears below is actually the function you want the derivative of.
  3. A step by step interactive solution will appear here.
  4. Press the right arrow to advance a step and the left arrow to go back.
  5. Follow along and make sure you understand each step.
  6. Enter another function and start again.

Features

Check the audio box to get audible information from each step.

I don't want to use x

Enter the variable you want to differentiate with respect to here. Note that for variables other than x your input must have correct syntax. There will be no attempts to add missing '*' or parentheses.

The function is not the one I entered

Make sure that the function you entered is what you wanted to enter. Typos occur. Also, parentheses are often in the wrong place or missing. Be careful when division is involved and exponents.
